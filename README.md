# wx-baisi
小程序-百思不得姐仿写

**自己学习微信小程序的时候仿照大神写的**
- 在视频和音频播放逻辑部分做了改进，不会在视频A播放的时候播放视频B，在播放A失效，音频同理
- 对app.json做出了部分改动，看起来更美观一些
- 对个人页面的菜单列表做了一些改进，让列表对齐，不会因为字多导致排版出错
